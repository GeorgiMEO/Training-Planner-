# Training Planner — План тренировок

**Автор:** Черский Георгий

## Описание

GUI‑приложение для планирования и учёта тренировок. Позволяет добавлять, просматривать и фильтровать тренировки, сохраняет данные в JSON‑файл между запусками.

## Функциональность

* Добавление тренировок с указанием даты, типа и длительности.
* Валидация ввода данных:
  * проверка формата даты (ГГГГ‑ММ‑ДД);
  * проверка, что длительность — положительное число.
* Фильтрация данных:
  * по типу тренировки;
  * по диапазону дат.
* Сохранение и загрузка данных из `trainings.json`.
* Удобный графический интерфейс на Tkinter.

## Требования

* Python 3.6 или выше.
* Стандартные библиотеки Python (не требуется установка дополнительных пакетов).
