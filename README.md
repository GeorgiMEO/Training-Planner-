# Training Planner

## Автор: [Черсский Георгий]
## Описание:
Графическое приложение для планирования и учета тренировок. Позволяет добавлять записи о тренировках с указанием даты, типа и длительности. Реализована фильтрация по типу и дате. Данные сохраняются в формате JSON.

## Технологии:
- Python 3.x
- Tkinter (стандартная библиотека)
- JSON

## Использование:
1. Запустите `main.py`.
2. Введите данные о тренировке и нажмите «Добавить тренировку».
3. Используйте фильтры для поиска записей.
4. Данные сохраняются автоматически в `data.json`.

## Примеры тестов:
1. Добавить тренировку с датой `2026-05-04`, типом `Бег`, длительностью `45`.
2. Попробовать ввести некорректную дату (например, `04.05.2026`) — появится ошибка.
3. Попробовать ввести отрицательную длительность — появится ошибка.
4. Отфильтровать по типу «Бег» — в таблице останутся только беговые тренировки.
