# Training Planner — План тренировок

**Автор:** Иванов Иван

## Описание

Приложение для планирования и учёта тренировок с возможностью фильтрации по типу и дате. Данные сохраняются в JSON‑файл между запусками программы.

## Функциональность

* Добавление тренировок с указанием даты, типа и длительности.
* Валидация ввода данных (формат даты, положительная длительность).
* Фильтрация записей по типу тренировки и диапазону дат.
* Сохранение данных в JSON‑формате.
* Удобный графический интерфейс на базе Tkinter.

## Требования

* Python 3.6+
* Стандартные библиотеки: `tkinter`, `json`, `os`, `datetime`
